# 第一阶段验收清单

## 1. 使用说明

本清单用于判断第一阶段是否真正完成。所有状态默认是 `TODO`，只有通过真实或样例 Excel / CSV 文件验证后，才能改为 `DONE`。

验收原则：

- 文件能上传，不等于项目完成。
- 图表能显示，不等于数据可信。
- 指标能计算，不等于经营判断正确。
- 字段含义不明时，必须标记风险。

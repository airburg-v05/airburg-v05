# V0.3D 稳定版基线

更新时间：2026-06-20

## 版本定位

V0.3D 是天猫四源本地数据分析的稳定基线版本。当前版本以手动上传真实报表、本地解析、本地安全聚合和浏览器端看板展示为核心，不包含后端、数据库、AI 或自动平台接口。

## 已包含能力

1. 四源上传工作台
2. 首页真实数据
3. 宝贝看板真实数据基础版
4. 店铺看板真实数据基础版
5. 系列分组管理基础版
6. 系列真实指标汇总基础版
7. 本地结果损坏校验
8. 售后隐私隔离

## 当前暂不包含

1. 趋势图
2. 目标设置
3. MTD/DLY
4. AI 分析
5. 数据库
6. 后端接口
7. 多店铺
8. 用户权限
9. 自动平台 API

## 本地存储 Key

1. `airburg_tmall_analysis_v2`
2. `airburg_tmall_series_groups_v1`

## 当前数据边界

1. 四源分析结果只保存安全聚合数据。
2. 售后订单、联系方式、地址、备注等敏感明细不进入长期本地结果。
3. 系列分组只保存系列名称、描述、商品 ID、创建时间和更新时间。
4. 系列指标按当前四源安全聚合结果实时匹配，不复制商品名称或原始报表行。
5. 当前阶段不接天猫 API，不做爬虫，不做自动登录，不接 AI 服务。

## 浏览器验收记录

使用同一份四源真实报表完成浏览器验收：

1. 上传页四源识别成功，结果已保存。
2. 首页可读取真实四源结果，当前经营日期为 `2026-06-18`。
3. 宝贝看板可读取真实四源结果。
4. 店铺看板可读取真实四源结果。
5. 系列看板可读取同一份真实四源结果。
6. 已创建 `P3系列` 和 `P1系列` 两个测试系列。
7. 系列刷新后仍可恢复。
8. 系列切换后指标同步更新。
9. 系列编辑后指标同步更新。
10. 无推广商品显示 `暂无推广数据`，推广花费和推广 ROI 显示 `--`。
11. 商品 ID 不逐字换行。
12. 表格横向滚动限制在表格容器内。
13. 390px 级别移动宽度未发现整页横向溢出。
14. 浏览器页面未发现新的 `NaN`、`Infinity`、`undefined` 展示。

## 冻结说明

V0.3D 之后如继续开发，应以本文件作为稳定基线，不在后续小修中改动：

1. 四源解析规则
2. GB18030 解码
3. 商品、计划、售后关联规则
4. 正式指标口径
5. `airburg_tmall_analysis_v2` 存储结构
6. `airburg_tmall_series_groups_v1` 存储结构
